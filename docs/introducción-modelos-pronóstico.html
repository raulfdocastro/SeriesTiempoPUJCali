Primero graficamos con plot la tendencia de la serie de tiempo / xts para las variables a analizar en este apartado:

De la anterior gráfica se nota claramente, que aunque anteriormente no se encontraron datos faltantes / NA en el conjunto de datos seleccionados para la precipitación es notorio que entre marzo (mes 3) y agosto (mes 8) de 2019 se presentan de seguido registros de lluvia que son cero, aunque es de esperar que no lloverá todos los días del año en Bogotá, tampoco tiene sentido que se presenten cinco meses de sin registros de lluvia; lo anterior permite inferir que el sensor / pluviómetro presentó fallas en dichos meses y esto se confirma al revisar los datos en el dataframe que contiene ceros con un formato diferente (int 0 vs dbl 0.0) en el periodo en mención1.

4.2 Modelo Holt-Winters

Ahora, aplicamos el modelo Holt-Winters para la serie mensual de temperatura3 en Medellín obtenida en el presente capítulo.

# Holt Winters log temperatura mensual Medellín
BOG_rain_2018_2019_monthly_HW <- HoltWinters(BOG_rain_2018_2019_monthly_ts, 
                                             seasonal = "additive")

plot(BOG_rain_2018_2019_monthly_HW, 
     main='Ajuste con Holt-Winters Precipitación total acumulada mensual Bogotá 2018-2019',
     xlab='Fecha', ylab='log(Temperatura [°C])',
     col="lightgreen")

# Holt Winters log temperatura mensual Medellín
MED_temp_2022_2023_monthly_HW <- HoltWinters(log(MED_temp_2022_2023_monthly_ts), 
                                             seasonal = "additive")

plot(MED_temp_2022_2023_monthly_HW, 
     main='Ajuste con Holt-Winters Temperatura máxima promedio mensual Medellín 2022-2023',
     xlab='Fecha', ylab='log(Temperatura [°C])',
     col="lightgreen")

A continuación se presentan la descomposición para el modelo obtenido anteriormente:

# Descomposición Holt Winters log temperatura Medellín
plot(fitted(MED_temp_2022_2023_monthly_HW), 
     main='Descomposición con Holt-Winters Temperatura máxima promedio mensual Medellín 2022-2023',
     xlab='Fecha', ylab='log(Temperatura [°C])',
     col="lightgreen")

4.3 Predicciones iniciales

Ahora que contamos con los modelos Holt Winters aplicados a la series mensuales de temperatura en Medellín, se procede a realizar unas predicciones iniciales para seis meses.

# Predicción Holt Winters log temperatura Medellín
MED_temp_2022_2023_monthly_Pred <- predict(MED_temp_2022_2023_monthly_HW, 6, prediction.interval = TRUE)
MED_temp_2022_2023_monthly_Pred

# Gráfico Predicción Holt Winters log temperatura Medellín
plot(MED_temp_2022_2023_monthly_HW, MED_temp_2022_2023_monthly_Pred, 
     main='Predicción con Holt-Winters Temperatura máxima promedio mensuañ Medellín 2022-2024',
     xlab='Fecha', ylab='log(Temperatura [°C])',
     col=c("lightgreen", "green"))

Las predicciones encontradas con Holt-Winters siguen una tendencia creciente de la temperatura máxima mensual y parece no tener en cuenta el ciclo climático, pero es una primera aproximación que como todo prónostico se debe analizar en profundidad y tener claro los datos usados para generar predicciones futuras.

  1. Los registros continuos de “ceros” van exactamente desde 2019-03-19 hasta 2019-08-07.↩︎

  2. Debido a la presencia de ceros en los datos de precipitación no se aplica el logaritmo a dicha variable para evitar resultados infinitos / INF.↩︎

  3. Como se anotó anteriormente, al aplicar al modelo HoltWinters a la precipitación mensual con registros de cero se obtiene error NA/NaN/Inf con lm.fit↩︎
